== DESCRIPTION

ZZZ is a liblary for issuing a certificate.

- Issuance of certificate and CRL
- Generating a CSR


== INSTALL

  $ git clone git://github.com/Hexa/ZZZ.git
  $ cd ZZZ
  $ rake build
  $ gem install pkg/ZZZ-<version>.gem


== REQUIREMENTS

- ruby 1.9.3
